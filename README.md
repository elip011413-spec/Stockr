# Stockr — AI-powered stock lookup app

Search any publicly traded company and get instant AI-analyzed stock info.

## How to deploy
1. Upload all files to your GitHub repository
2. Go to Settings → Pages → Branch: main → Save
3. Your app will be live at https://yourusername.github.io/your-repo-name

## Files
- `index.html` — the entire app (upload this to GitHub)
